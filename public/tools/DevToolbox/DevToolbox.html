            API & Integrations

            JSONPath/XPath, HTTP runner locale, utility cURL e validazione schema JSON.

            JSONPath Tester Estrai nodi da payload JSON (dot/bracket/wildcard)
            JSON Input
            Result
            Guida rapida + esempi

            Uso: inserisci JSON a sinistra e una query JSONPath nel campo in alto. Supporta ., ["key"], [index], [start:end], [*].

            Esempio JSON:
{"items":[{"id":1,"name":"a"},{"id":2,"name":"b"}]}

Query:
$.items[*].id

Output atteso:
[1,2]
            XPath Tester (XML) Valuta XPath su XML e mostra i nodi trovati
            XML Input
            Result
            Guida rapida + esempi

            Uso: incolla XML e scrivi un XPath valido. Mostra nodi/valori trovati.

            Esempio XML:
<root><book id="1"><title>Dev</title></book></root>

XPath:
//book[@id='1']/title/text()

Output:
Dev
            HTTP Request Runner Esegui GET/POST/... via backend locale (evita limiti CORS del browser)
            Headers
            Body
            Guida rapida + esempi

            Uso: scegli metodo, URL, headers/body e premi Send Request. Il backend locale esegue la chiamata (utile contro blocchi CORS browser).

            Esempio rapido:
Method: GET
URL: https://httpbin.org/get
Headers: Authorization: Bearer abc123

Premi: Send Request
            cURL Builder Genera cURL da form, oppure fai parse di un comando cURL esistente
            Headers
            Body
            Guida rapida + esempi

            JSON Schema Validator Verifica payload JSON contro schema (type, required, properties, items, enum...)
            Schema JSON
            Payload JSON
            Guida rapida + esempi
